# 🏦 Banking Management System

A desktop-based **Banking Management System** developed in **Java** using **Java Swing**. This project demonstrates the implementation of Object-Oriented Programming (OOP) principles while providing a graphical interface for managing clients, accounts, and banking transactions.

---

## 📖 Overview

The Banking Management System is designed to simulate the core functionalities of a real-world banking application. It allows users to register clients, create different types of bank accounts, perform transactions, and securely store banking records using file handling.

The application features a clean Java Swing interface, making it easy to manage banking operations through an interactive desktop application.

---

## ✨ Features

### 👤 Client Management
- Register new clients
- Store personal information (Name, CNIC, Phone Number)
- Remove clients and all associated accounts

### 💳 Account Management
- Create Savings Accounts
- Create Current Accounts
- Automatic Account Number Generation
- Associate multiple accounts with a client

### 💰 Banking Transactions
- Deposit money
- Withdraw money
- Display current account balance

### 🏦 Savings Account
- Interest Rate Support
- Interest Calculation
- Minimum balance validation (Rs. 500)

### 💼 Current Account
- Overdraft Limit Support
- Withdraw beyond available balance within overdraft limit

### 📊 Dashboard
- Total Registered Clients
- Active Accounts
- Total Bank Funds
- Client Summary Table

### 💾 Data Persistence
- Automatic saving of client and account data
- Automatic loading of saved data on application startup
- Data stored using text file handling
